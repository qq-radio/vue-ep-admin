# BasicImageUpload 图片上传

## 基本使用

:::demo
basic-image-upload/basic
:::

## 单张图片大小限制

您可以传入`max-size`限制单张图片的大小，`max-size`默认为2，单位为MB

:::demo
basic-image-upload/max-size
:::

## 最大允许上传数量

您可以传入`limit`限制最大允许上传数量，`limit`默认为1

:::demo
basic-image-upload/limit
:::

## 透传预览图片属性

当组件禁用时，隐藏上传图标，仅可预览图片，您可以通过`previewImageProps`传递预览图片组件透传属性

:::demo
basic-image-upload/preview-image-props
:::
